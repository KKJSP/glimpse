# <img src="https://raw.githubusercontent.com/KKJSP/glimpse/refs/heads/main/icons/favicon64.png" alt="drawing" style="height: 1.2em; vertical-align: bottom; margin-bottom:0.1em;"/> Glimpse

A lightweight local video monitoring system built with WebRTC entirely using Gemini.

## How to use
1. Go to [Glimpse](https://kkjsp.github.io/glimpse) on both your **Camera** and **Monitor** devices.
2. Enter a code on one of the devices and click on Monitor.
3. Enter the same code on the other device and click on Camera.
4. The session should begin immediately, allowing you to see the camera feed on the monitor device.

**Note**: Both devices must be connected to the same Wi-Fi network for the local video streaming to work.

## Features
- **Local Monitoring:** Video travels directly between devices on your Wi-Fi.
- **Motion Detection:** Visual and audio alerts with a 30-second setup grace period.
- **Manual Ring:** Send an alert from the camera to the viewer.
