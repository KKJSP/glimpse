# Glimpse

A lightweight local video monitoring system built with WebRTC entirely using Gemini.

## How to use
1. Host these files on GitHub Pages.
2. Open the main URL on your **Viewer** device (e.g., Mac).
3. Scan the generated QR code with your **Broadcaster** device (e.g., iPad).
4. The monitor session will start automatically once the camera is connected.

## Features
- **Local Monitoring:** Video travels directly between devices on your Wi-Fi.
- **Motion Detection:** Visual and audio alerts with a 30-second setup grace period.
- **Privacy First:** Uses secure hash fragments for connection IDs.
- **Manual Ring:** Send an alert from the camera to the viewer.
